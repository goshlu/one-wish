# 更新日志

## [1.1.0] - 2025-11-24

### Added ✨

- **MyAutoComplete** 自动完成组件
  - 支持键盘导航（↑/↓/Enter/Escape）
  - 支持自定义选项标签
  - 支持远程搜索
  - 支持清除按钮
  - 自动过滤匹配选项
  - 无匹配时显示空状态提示

- **MyCascader** 级联选择器
  - 支持多层级数据选择
  - 支持清空功能
  - 支持禁用选项
  - 现代化的下拉面板设计
  - 流畅的交互体验

- **MyCheckbox** 多选框组件
  - 支持单个多选框
  - 支持多选框组
  - 支持半选状态
  - 支持禁用状态
  - 优雅的选中动画

- **MyColorPicker** 颜色选择器
  - 支持十六进制颜色输入
  - 支持预定义颜色
  - 支持透明度背景显示
  - 点击外部自动关闭
  - 现代化的面板设计

### Improved 🔧

- 重新设计所有新组件样式，采用 Ant Design 5.0 设计风格
- 统一的颜色系统和圆角规范
- 改进的阴影和过渡动画效果
- 优化组件的交互体验
- 完善所有组件的 TypeScript 类型定义

### Documentation 📚

- 为所有新组件添加完整的文档和示例
- 添加使用场景说明
- 添加键盘操作说明
- 更新使用指南，包含所有新组件示例
- 添加 TypeScript 类型导入说明

## [1.0.0] - 2025-11-24

### Added ✨

- **MyButton** 按钮组件
  - 支持 6 种类型：primary、success、warning、danger、info、default
  - 支持 3 种尺寸：large、default、small
  - 支持朴素按钮、圆角按钮
  - 支持禁用和加载状态
- **MyInput** 输入框组件
  - 支持多种类型：text、password、number、email、tel
  - 支持 3 种尺寸：large、default、small
  - 支持清空功能
  - 支持字数限制和统计
  - 支持禁用和只读状态
  - 完整的事件支持（input、change、focus、blur、clear）

### Features 🎉

- ✅ 基于 Vue 3.5 + TypeScript 5.7 + Vite 5.4
- ✅ 完整的 TypeScript 类型定义
- ✅ 支持 ES Module 和 UMD 两种格式
- ✅ 支持 Tree-shaking 按需引入
- ✅ 基于 CSS 变量的主题系统
- ✅ 简洁现代的设计风格
- ✅ 完善的文档和示例
