# 更新日志

## [1.1.0] - 2025-11-24

### Added ✨

- **MyFloatButton** 悬浮按钮组件
  - 支持圆形和方形两种形状
  - 支持四个角落位置定位
  - 支持徽标显示（数字或文字）
  - 支持描述文字
  - 支持作为链接使用
  - 支持禁用状态
- **MyFloatButtonGroup** 悬浮按钮组组件

  - 支持点击和悬停两种触发方式
  - 支持多个悬浮按钮组合
  - 简洁的展开/收起动画

- **VitePress 文档系统**
  - 完整的组件文档
  - 在线示例演示
  - API 参考文档
  - 使用指南
  - TypeScript 支持文档

### Improved 🔧

- 优化了组件的类型定义
- 完善了 TypeScript 导出

## [1.0.0] - 2025-11-24

### Added ✨

- **MyButton** 按钮组件
  - 支持 6 种类型：primary、success、warning、danger、info、default
  - 支持 3 种尺寸：large、default、small
  - 支持朴素按钮、圆角按钮
  - 支持禁用和加载状态
- **MyInput** 输入框组件
  - 支持多种类型：text、password、number、email、tel
  - 支持 3 种尺寸：large、default、small
  - 支持清空功能
  - 支持字数限制和统计
  - 支持禁用和只读状态
  - 完整的事件支持（input、change、focus、blur、clear）

### Features 🎉

- ✅ 基于 Vue 3.5 + TypeScript 5.7 + Vite 5.4
- ✅ 完整的 TypeScript 类型定义
- ✅ 支持 ES Module 和 UMD 两种格式
- ✅ 支持 Tree-shaking 按需引入
- ✅ 基于 CSS 变量的主题系统
- ✅ 简洁现代的设计风格
- ✅ 完善的文档和示例
