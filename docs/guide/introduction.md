# 介绍

One Wish 是一个基于 Vue 3 + TypeScript + Vite 构建的现代化组件库。

## 设计理念

- **简洁至上** - 去除冗余视觉效果，专注核心功能
- **性能优先** - 基于 Vite 构建，极速开发体验
- **类型安全** - 完整的 TypeScript 支持
- **按需加载** - 支持 Tree-shaking，优化包体积

## 核心特性

| 特性        | 说明                                    |
| ----------- | --------------------------------------- |
| 🚀 快速构建 | Vite 5.x 提供极速的开发和构建体验       |
| 💪 类型安全 | TypeScript 编写，完整类型定义和智能提示 |
| 🎨 简洁设计 | 现代化 UI，无冗余动画，响应迅速         |
| 📦 按需引入 | 支持 Tree-shaking，只打包使用的组件     |
| 🌈 主题定制 | 基于 CSS 变量，可灵活定制主题           |
| 📘 完善文档 | 详细的 API 文档和在线示例               |
