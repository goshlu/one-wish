# Release v1.1.0

## 🎉 新增功能

### 表单组件

#### AutoComplete 自动完成
- ✨ 支持键盘导航（↑/↓/Enter/Escape）
- ✨ 支持自定义选项标签
- ✨ 支持远程搜索
- ✨ 支持清除按钮
- ✨ 自动过滤匹配选项
- ✨ 无匹配时显示空状态提示

#### Cascader 级联选择器
- ✨ 支持多层级数据选择
- ✨ 支持清空功能
- ✨ 支持禁用选项
- ✨ 现代化的下拉面板设计
- ✨ 流畅的交互体验

#### Checkbox 多选框
- ✨ 支持单个多选框
- ✨ 支持多选框组
- ✨ 支持半选状态
- ✨ 支持禁用状态
- ✨ 优雅的选中动画

#### ColorPicker 颜色选择器
- ✨ 支持十六进制颜色输入
- ✨ 支持预定义颜色
- ✨ 支持透明度背景显示
- ✨ 点击外部自动关闭
- ✨ 现代化的面板设计

## 🔧 改进

### 设计系统
- 🎨 采用 Ant Design 5.0 设计风格
- 🎨 统一的颜色系统（rgba 透明度）

- 🎨 改进的阴影效果
- 🎨 流畅的过渡动画

### 用户体验
- ⚡ 优化组件交互体验
- ⚡ 改进键盘导航支持

- ⚡ 统一的禁用状态样式

### TypeScript
- 📝 完善所有组件的类型定义

- 📝 改进类型推断
